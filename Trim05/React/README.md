# Frontend (React) — Guía de inicio

## Requisitos
- Node.js 18+

## Instalación
1. Desde `React/`, instala dependencias:
   - `npm install`

## Ejecución en desarrollo
- `npm start`
- Abre `http://localhost:3000`

Proxy:
- El `package.json` apunta a `http://localhost:5001` para API (`proxy`). Asegúrate de que el backend esté levantado.

## Build de producción
- `npm run build`
- Genera la carpeta `build/` lista para servir.
